## Solution

```java
import java.util.List;

class Solution {
    public static int differenceOfSum(int[] nums) {
        int elementSum = 0;
        int digitSum = 0;

        for (int i = 0; i < nums.length; i++) {
            digitSum += nums[i];

            if (nums[i] / 10 > 0) {
                int tmp = nums[i];
                while (tmp != 0) {
                    elementSum += tmp % 10;
                    tmp /= 10;
                }
                continue;
            }
            elementSum += nums[i];
        }

        System.out.println("element: " + elementSum);
        System.out.println("digit: " + digitSum);
        return Math.abs(elementSum - digitSum);
    }
}
```
