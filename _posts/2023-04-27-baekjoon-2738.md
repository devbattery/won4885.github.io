- [Reference](https://www.acmicpc.net/problem/2738)
- N, M 헷갈려서 계속 런타임 에러 떴음

<br>

## Solution

```python
N, M = map(int, input().split())

l1 = []
for _ in range(N):
    tmp1 = list(map(int, input().split()))
    l1.append(tmp1)

l2 = []
for _ in range(N):
    tmp2 = list(map(int, input().split()))
    l2.append(tmp2)

for i in range(N):
    for j in range(M):
        print(l1[i][j] + l2[i][j], end=' ')
    print()
```
