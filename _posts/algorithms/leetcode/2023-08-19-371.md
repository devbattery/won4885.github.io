- 전가산기 구현 문제
    - 미디움이 아니라 하드 같음..
    - 이해 너무 힘들어서 그냥 `sum()` 썼음..

<br>

## Solution

```python
class Solution:
    def getSum(self, a: int, b: int) -> int:
        return sum([a, b])
```
